一、重力

（一）重力的产生

先来看重力的定义：由于地球的吸引而使物体受到的力叫做重力，从定义中能直接看出，重力的产生原因是地球的吸引。

需要牢记这几个注意事项：

  1. 重力的施力物体是地球，受力物体为地面附近的一切物体；
  2. 物体所受的重力与它所处的运动状态无关，静止或运动状态下，重力均存在且大小不变；
  3. 不能说重力就是地球对物体的吸引力，该知识点会在必修二详细解释，可先做好标记。

（二）重力大小

接下来学习重力大小的计算，核心公式必须掌握：物体所受重力G与物体的质量m的关系式为G = mg。

关于公式的关键知识点梳理：

  1. 重力的单位是牛顿，简称牛，符号用N表示；
  2. 公式中的g是自由落体加速度，大小为g = 9.8 N/kg，粗略计算时可取10N/Kg；
  3. g有两个等价单位，可以是N/Kg，也可以是m/s²，且1N/Kg=1m/s²。

思考

结合所学内容，尝试回答以下问题：

  1. 用何种仪器测量物体的重力呢？是用天平还是测力计？
  2. 如何用测力计测量物体的重力？
  3. 可以说测力计对物体的拉力大小等于物体的重力，能说成测力计对物体的拉力就是物体的重力吗？为什么？

标准答案：

  1. 应该用测力计（弹簧测力计）测量物体的重力，天平是用来测量物体质量的。
  2. 用测力计测量物体重力的方法：将物体挂在测力计的挂钩上，待测力计示数稳定后，读取测力计的示数，这个示数就是物体所受的重力大小。
  3. 不能说测力计对物体的拉力就是物体的重力。虽然测力计对物体的拉力大小等于物体的重力，但这两个力的性质不同：重力是由于地球的吸引而使物体受到的力，施力物体是地球；而测力计对物体的拉力是弹力，施力物体是测力计。只能说它们大小相等，不能说它们是同一个力。

（三）重力的方向

观察自由下落的物体下落方向和静止悬挂物体绳子的方向，可总结出重力的核心方向：竖直向下。

图：自由落体运动

这里有两个高频易错点，务必区分清楚：

  1. 竖直向下不能说垂直向下，竖直向下是垂直于水平面的方向，垂直向下会随接触面的变化而改变；
  2. 竖直向下不能说指向地心，仅地球的两极和赤道处，重力的方向才指向地心。

（四）重心

物体的各部分都会受到重力作用，为了方便研究，引入重心的概念：从效果上看，物体各部分受到的重力可以集中作用于一点，这一点叫做物体的重心。

重心的位置分布有以下规律，是常考知识点：

  1. 形状规则、质量均匀分布的物体的重心，位置在其几何中心；
  2. 质量分布不均匀的物体，重心的位置除了跟物体的形状有关外，还跟物体内质量分布有关；
  3. 重心的位置可以在物体上，也可以在物体外，比如圆环的重心在其圆心处，而非圆环本身。

总结与思考

通过以上对重力的学习，梳理并回答：

  1. 描述物体所受的重力，需要哪几个重要要素？
  2. 根据初中所学知识，在物理的学习中通过什么方式体现出这三个要素呢？

标准答案：

  1. 描述物体所受的重力，需要三个重要要素：大小、方向、作用点。
  2. 在物理的学习中，通过力的图示来体现力的三个要素：用一条有向线段来表示力，线段的长度表示力的大小，箭头的方向表示力的方向，起点（或终点）表示力的作用点。

（五）力的图示

结合上述思考的力的三要素，物理中用特定方式直观表示力：用一条有向线段来表示力的三要素，这种方法就是力的图示，具体对应关系为：

线段的长度表示力的大小，箭头的方向表示力的方向，起点（或终点）表示力的作用点。

力的图示的作图步骤，需按顺序规范操作：

  1. 选定合适的标度，标度一旦确定，作图过程中不可随意更改；
  2. 从作用点开始沿力的方向画一线段，线段的长度由力的大小和所选标度确定；
  3. 在线段的末端画上箭头，箭头表示力的方向。

学完重力，接下来认识第二种常见的力——弹力，学习前先掌握基础概念：形变。

二、弹力

（一）形变

观察物体在力的作用下的变化，可得出形变的定义：物体在力的作用下形状或体积发生的改变，叫做形变。

图：物体受力发生形变

思考

是否任何物体受到力的作用，都会发生形变呢？

标准答案：

是的，任何物体受到力的作用，都会发生形变。有些形变肉眼可以直接观察到（如弹簧被拉伸），有些形变非常微小，肉眼难以直接观察到（如桌面被重物压时产生的微小形变）。微小形变可以通过放大装置或光学方法观察到，比如用激光照射桌面，观察反射光的变化，就能检测到桌面的微小形变。

思路与讨论

带着以下问题，继续学习后续的弹力知识：

  1. 发生形变的物体，会对使它发生形变的物体产生什么作用效果呢？
  2. 它产生这样效果的原因是什么？

标准答案：

  1. 发生形变的物体，会对使它发生形变的物体产生力的作用效果，这种力就是弹力。比如弹簧被拉伸时，弹簧会对拉伸它的手产生拉力。
  2. 产生这种效果的原因是：发生弹性形变的物体由于要恢复原状，对与它接触的物体产生力的作用。这是物体的一种固有属性，叫做弹性。

（二）弹力

结合上述思考，理解弹力的定义：发生弹性形变的物体由于要恢复原状，对与它接触的物体产生力的作用，这种力叫做弹力。

从定义中提炼弹力的产生条件，这是判断弹力是否存在的关键，两个条件缺一不可：

第一，物体间相互接触（互相挤压、拉伸或扭曲）；第二，接触的物体间发生弹性形变。

弹力的施力物体

弹力的施力物体有明确判定依据：发生弹性形变的物体，受力物体则是使它发生形变的物体。

结合实例理解，思考以下问题：

用手拉弹簧、用手压弹簧时，手受到弹簧的作用力中，谁是施力物体？

图：手拉弹簧

标准答案：

手受到弹簧的作用力中，施力物体是弹簧。因为弹簧发生了弹性形变，要恢复原状，所以对与它接触的手产生了力的作用。根据弹力的定义，发生弹性形变的物体是弹力的施力物体，使它发生形变的物体是受力物体。

弹力的方向

先思考这个问题：手受到弹簧的弹力作用，弹力的方向与施力物体弹簧恢复形变的方向和形变的方向有什么关系？

由此总结弹力方向的核心规律：弹力的方向与施力物体恢复形变的方向相同，与形变的方向相反。

结合规律，掌握两种常见的弹力方向：

  1. 绳的拉力方向：沿绳指向绳收缩的方向，可尝试在对应图示中画出绳提供弹力的方向；
  2. 杆的弹力方向：不一定沿杆方向，要根据杆的形变情况判断，可尝试在对应图示中画出杆提供弹力的方向。
弹力有无的判断方法

掌握两种实用的判断方法，应对不同场景：

  1. 根据弹力的产生条件直接判断法，适用于形变明显的情况；
  2. 假设法，常用于微小形变的情况，具体操作：假设接触面不存在，观察物体的运动状态是否改变，以此判断弹力是否存在。

实例练习

斜面光滑，分析左右两图中的小球与斜面之间是否存在弹力？可尝试用假设法解题。

标准答案：

用假设法判断：

  1. 左图：假设斜面不存在，小球会竖直下落，说明斜面对小球有支持作用，因此存在弹力。
  2. 右图：假设斜面不存在，小球仍会静止在水平面上，说明斜面对小球没有支持作用，因此不存在弹力。

结论：左图中小球与斜面之间存在弹力，右图中小球与斜面之间不存在弹力。

三、胡克定律

弹簧的弹力是最常见的弹力类型，思考与讨论：

弹簧在形变时产生的弹力与弹簧的伸长量是有关系的，那么，弹簧在弹性限度内，弹力大小与其伸长量有什么具体关系呢？

带着这个问题，学习探究弹簧弹力与形变量的实验及胡克定律。

（一）探究弹簧弹力与形变量的关系

图：探究弹簧弹力与形变量的关系实验
1. 实验思路

先明确实验的核心问题，理清探究思路：

  1. 弹簧弹力F如何测量？
  2. 弹簧的伸长量x如何测量？
  3. 如何能直观找出弹簧弹力与形变量的关系？
2. 实验步骤

按以下步骤规范进行实验，注意操作细节：

  1. 按图示安装实验装置，记下弹簧下端不挂钩码时所对应的刻度l₀（弹簧原长）；
  2. 在弹簧下端悬挂一个钩码，平衡时记下弹簧的总长度和钩码所受的重力，此时钩码重力大小等于弹簧弹力大小；
  3. 增加钩码的个数，重复上述实验过程，将数据填入表格，以F表示弹力，l表示弹簧的总长度，x＝l－l₀表示弹簧的形变量；
  4. 根据实验数据描点、作图，分析弹力与形变量的关系。

实验可得出明确结论：弹簧发生弹性形变时，弹力F的大小跟弹簧伸长（或缩短）的长度x成正比。

3. 实验注意事项

为减小实验误差，保证实验结果准确，需注意以下几点：

  1. 所挂钩码不要过重，以免弹簧被过分拉伸，超出其弹性限度，超出弹性限度后，弹力与形变量不再成正比；
  2. 每次所挂钩码的质量差尽量大一些，从而使坐标上描的点尽可能稀一些，这样作出的图像更精确；
  3. 测弹簧长度时，一定要在弹簧竖直悬挂且处于稳定状态时测量，刻度尺要保持竖直并靠近弹簧，以免增大读数误差。

（二）胡克定律

基于上述实验结论，总结得出胡克定律的核心内容：在弹性限度内，弹簧发生弹性形变时，弹力的大小F跟弹簧伸长（或缩短）的长度x成正比。

胡克定律的核心公式：F＝kx。

关于公式中劲度系数k的关键知识点，需全面掌握：

  1. k为弹簧的劲度系数，单位是N/m；
  2. 劲度系数k只与弹簧本身有关，由弹簧本身的材料、长度、粗细、匝数等因素决定；
  3. k反映了弹簧的软硬程度，k越大，弹簧越硬，其长度越难改变。

为什么汽车的轮胎、鞋子和手套有很多花纹，这些花纹有什么作用吗？

标准答案：

汽车轮胎、鞋子和手套上的花纹是为了增加接触面的粗糙程度，从而增大摩擦力。这样可以提高轮胎与地面的附着力，防止打滑；增加鞋子和手套与接触面之间的摩擦，使人行走或抓握更加稳固。

初中我们已经学习过：两个相互接触的物体，当它们发生相对运动或具有相对运动的趋势时，就会在接触面上产生阻碍相对运动或相对运动的趋势的力，这种力就叫作摩擦力。

思考与讨论

用弹簧测力计拖动水平固定木板上的木块，使它做匀速运动，测力计的示数等于木块所受摩擦力的大小。改变木块和木板之间的压力，摩擦力的大小也随之改变。如果摩擦力的大小跟压力的大小存在某种定量关系的话，它们可能是怎样的关系呢？

标准答案：

滑动摩擦力的大小跟压力成正比。即当接触面粗糙程度不变时，压力越大，滑动摩擦力越大，它们之间存在正比关系。数学表达式为：Ff = μFN，其中FN是压力，μ是动摩擦因数。

  1. 接触面粗糙；
  2. 物体间相互接触且挤压；

思考

两物体间的压力是否一定等于重力大小？

标准答案：

不一定。压力是指两个相互接触的物体之间的相互作用力，其大小不一定等于重力。只有当物体放在水平面上时，压力才等于重力。如果物体放在斜面上，压力小于重力；如果物体被向下压，压力大于重力；如果物体被向上拉，压力可能为零。

2.公式：Ff = μFN

μ是动摩擦因数，它的数值只跟相互接触的两个物体的材料和接触面的粗糙程度有关；动摩擦因数无单位。

【例题】

在我国东北寒冷的冬季，有些地方用雪橇作为运输工具。一个有钢制滑板的雪橇，连同木料的质量为4.9×103 kg。在水平的冰道上，马要在水平方向上用多大的力，才能够拉着雪橇匀速前进？

解析：

雪橇在水平方向上匀速前进，受力平衡。

雪橇的质量：m = 4.9×103 kg

重力：G = mg = 4.9×103 × 9.8 = 4.802×104 N

在水平冰道上，压力等于重力：FN = G = 4.802×104 N

查表得钢—冰的动摩擦因数：μ = 0.02

滑动摩擦力：Ff = μFN = 0.02 × 4.802×104 = 960.4 N

由于雪橇匀速前进，马的拉力与摩擦力平衡，所以拉力大小等于摩擦力大小。

答案：马要在水平方向上用约960 N的力。

二、牛顿第三定律

（一）实验探究：用弹簧测力计探究作用力和反作用力的关系

我们可以通过图3.3-3所示的装置进行实验，把A、B两个弹簧测力计连接在一起，B的一端固定，用手拉测力计A。可以看到两个测力计的指针同时移动。这时，测力计B受到A的拉力F，测力计A则受到B的拉力F'。F与F'有什么关系呢？

（二）实验现象

  1. 实验现象：用手拉弹簧秤A，可以看到两个弹簧秤的指针同时运动，且两个弹簧秤的示数是相等的，改变手拉弹簧的拉力，弹簧秤的示数也随着改变，但两个示数总相等。
  2. 实验结论：作用力和反作用力大小总是相等的，也是同时变化的。

（三）牛顿第三定律

  1. 内容：两个物体之间的作用力和反作用力总是大小相等，方向相反，作用在一条直线上。
  2. 公式：F = -F'（负号表示作用力与反作用力的方向相反）

牛顿第三定律的适用范围

牛顿第三定律是个普遍规律。所阐明的作用力与反作用力的关系不仅适用于静止的物体之间，也适用于相对运动的物体之间，这种关系与作用力的性质、物体质量大小、作用方式（是否接触）、物体运动状态及参考系的选择均无关。

对牛顿第三定律的进一步理解

  1. 作用力和反作用力的三个特征：①等值；②反向；③共线。
  2. 作用力和反作用力的四个性质：①异体性；②同时性；③相互性；④同质性。

一、合力和分力

思考与讨论：

（一）合力和分力

  1. 合力：假设一个力单独作用的效果跟某几个力共同作用的效果相同，这个力就叫做那几个力的合力。
  2. 分力：假设几个力共同作用的效果跟某个力单独作用的效果相同，这几个力就叫作那个力的分力。

（二）合力和分力的关系

  1. 等效性：合力的作用效果与分力的共同作用效果相同，它们在效果上可以相互替代。"等效替代"
  2. 同体性：各个分力是作用在同一物体上的，分力与合力为同一物体，作用在不同物体上的力不能求合力。
  3. 瞬时性：各个分力与合力具有瞬时对应关系，某个分力变化了，合力也同时发生变化。

  2. 求合力大小：F = √(F₁² + F₂²) = √(45² + 60²) = √(2025 + 3600) = √5625 = 75N

  4. 方向描述：合力F与F₁成53°角，指向第一象限

答案：合力F的大小为75N，方向与F₁成53°角（竖直向上偏转）

思考与讨论：

  1. F₁、F₂大小一定，夹角增大，合力如何变化？合力什么时候最大，什么时候最小？合力的范围如何？

①大小为1N、2N、3N三力的合力最大值和最小值？

标准答案：

  1. 夹角增大，合力减小。当夹角为0°时合力最大（等于F₁+F₂），当夹角为180°时合力最小（等于|F₁-F₂|）。合力范围：|F₁-F₂| ≤ F ≤ F₁+F₂

  3. ①1N、2N、3N：最大值=1+2+3=6N，最小值=0（因为3=1+2）
    ②2N、2N、2N：最大值=2+2+2=6N，最小值=0（三力可平衡）
    ③2N、2N、5N：最大值=2+2+5=9N，最小值=1N（因为5-2-2=1）

（五）正交分解法

1.定义：

把力沿着两个选定的互相垂直的方向分解

2.正交分解的目的：

是把复杂的利用平行四边形定则进行力的合成转化为普通的代数运算，将力的合成化简为同向或反向或垂直方向。便于运用普通代数运算公式来解决力的合成。

图：正交分解法
3.正交分解的基本思想：

正交分解法求合力，运用了"欲合先分"的策略，即为了合成而分解，降低了运算的难度，是一种重要思想方法。

4.正交分解的步骤：
  1. 建立坐标系（通常取合力方向为x轴，垂直方向为y轴）
  2. 将各力分解到x轴和y轴上
  3. 分别求x轴和y轴上的合力：Fx合 = ΣFx，Fy合 = ΣFy
  4. 求合力大小：F = √(Fx合² + Fy合²)
  5. 求合力方向：tanθ = Fy合/Fx合
图：正交分解步骤

【例题】用正交分解法求合力

物体受到四个力的作用：F₁=10N水平向右，F₂=8N竖直向上，F₃=6N水平向左，F₄=4N竖直向下。求这四个力的合力。

解题步骤：

  1. 建立坐标系：水平方向为x轴，竖直方向为y轴
  2. 求x方向合力：Fx合 = F₁ - F₃ = 10 - 6 = 4N（向右）
  3. 求y方向合力：Fy合 = F₂ - F₄ = 8 - 4 = 4N（向上）
  4. 求合力大小：F = √(4² + 4²) = √32 = 4√2 ≈ 5.66N
  5. 求合力方向：与x轴夹角θ = 45°

答案：合力大小为4√2N，方向与水平方向成45°角，指向右上方

  2. 受力分析：重力G、支持力N、摩擦力f

三、整体法和隔离法

（一）整体法和隔离法

  1. 整体法：将运动状态相同的几个物体作为一个整体进行受力分析的方法。
  2. 隔离法：将研究对象与周围物体分隔开进行受力分析的方法。

（二）整体法和隔离法选用原则

  1. 整体法的选用原则：研究系统外的物体对系统整体的作用力，受力分析时不要再考虑系统内物体间的相互作用。
  2. 隔离法的选用原则：研究单个物体所受的作用力，一般隔离受力较少的物体。

（三）整体法和隔离法应用时受力分析的基本方法和步骤

  1. 明确研究对象：在进行受力分析时，研究对象可以是某一个物体，也可以是保持相对静止的若干个物体（整体）。研究对象确定以后，只分析研究对象以外的物体施予研究对象的力（既研究对象所受的外力），而不分析研究对象施予外界的力。
  2. 隔离研究对象，按顺序找力：把研究对象从实际情景中分离出来，按先已知力，再重力，再弹力，然后摩擦力（只有在有弹力的接触面之间才可能有摩擦力），最后其它力的顺序逐一分析研究对象所受的力，并画出各力的示意图。
  3. 只画性质力，不画效果力：画受力图时，只能按力的性质分类画力，不能按作用效果（拉力、压力、阻力等）画力，否则将出现重复。

解题步骤：

  1. 整体法：将a、b、c作为整体，受力分析：
    Fb=5N向右，Fc=10N向右

    水平方向：f₁对b向右 = f₂对b向左 = 5N
    所以f₂=5N

答案：C（f₁=0，f₂=5N，f₃=5N）
